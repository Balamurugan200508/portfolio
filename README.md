# Personal Portfolio

A professional, high-end scrollytelling personal portfolio built with modern web technologies. This project features dynamic scroll-linked image sequences, parallax text overlays, glassmorphic project grids, and a visually-driven tech stack section.

## Features

- **Dynamic Scrollytelling**: Engaging scroll-driven animations and image sequences using HTML5 Canvas.
- **Glassmorphic Design**: Modern, sleek aesthetics for the projects and tech stack sections.
- **Responsive Layout**: Fully responsive design that works seamlessly across all devices.
- **Animated Interactions**: Smooth entry and scroll animations powered by Framer Motion.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) 
- **Library**: [React](https://reactjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Balamurugan200508/portfolio.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
